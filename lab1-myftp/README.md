# lab1-myFTP
an implementation of a simplified FTP  
Here is [lab1 document](https://edu.n2sys.cn/#/tut_lab/lv1/README)
  
## Usage:
* `mkdir build && cd build && cmake .. && make` in root directory  
* `./ftp_server hostname port`
* `./ftp_client`

## Note:
Commands supported:
* open  
* auth
* ls
* get
* put
* quit
